# Dental Clinic Website

Publishable V1 for a Muzaffarnagar dental clinic website built with Next.js, React Three Fiber, and Three.js.

## Highlights

- Responsive public website with Hindi/English toggle
- Appointment request flow without login
- WhatsApp click-to-chat flows
- Contact information and Google Maps embed
- Interactive 3D dental anatomy hero with highlighted teeth, maxilla, mandible, and temporomandibular joint (TMJ)
- Placeholder clinic, doctor, service, review, and gallery content ready to replace with real details

## Local Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Verification

```bash
npm run lint
npm run build
```
